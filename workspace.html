User Manual

👋 Getting Started & Spawning

  • Categories: Browse Furniture, Electronics, Vehicles, Characters, and Props from the left sidebar to add them.
  • Search: Use the search box to quickly find specific models.

🖱️ Mouse & Keyboard Controls

  • Move: Click and hold an object to drag it around the scene.
  • Rotate: Select an object and use the Q and E keys to rotate it.
  • Properties: Double-click an object to open its property panel.

🎛️ Properties & Customization

  • Transform: Adjust the Object Height and Width.
  • Appearance: Change the Color or switch Materials (Wood, Plastic, Metal) for supported items.
  • Special: Adjust Lamp Intensity, or upload a custom Carpet Image (JPEG/PNG/WEBP).
  • Object Actions: Use DELETE, RESTORE, or STACK ON PREVIOUS.

🗣️ Voice & Text Commands

Use the microphone (🎙️) or text input to enter commands:

  • "Spawn a table", "Delete the chair", "Clear all objects"

⚙️ Workspace Tools

  • 💾 Save Project: Save your current scene.
  • 🔄 Reset Physics: Realign or reset any fallen objects.
  • 🧭 Compass: Toggle the compass to orient yourself in 3D space.
  • Live AR: Click the LIVE AR badge to view the scene in Augmented Reality.
